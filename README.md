# StockTwits Board

Enter in a stock symbol & display the latest messages from Stocktwis!

## Technologies Used

-ReactJS
-Express
-Stocktwits API
-Bootstrap

### Start App

To start this app:

1. clone this repo
2. "yarn install"
3. "cd yarn install"
4. "npm run dev"

The first two will install any packages needed, whereas the last command will the applicaton on "localhost:3000"
